Baseball score processing, part 2 (Using Arrays)

Problem

Add code to the program to print the box score of a completed game like a normal Boxscore. The first line of output will contain the runs scored in each inning by the visiting team, and the second line will contain the runs scored in each inning by the home team. The number of characters in each line should be the same, except that the box score of a game where the home team is winning at the start of the second half of the ninth inning should have nine characters in the first line and eight characters in the second. Assume that no more than nine runs are scored in any inning, and that the game ends after at most twenty innings. Do not include any blank characters in the output.
