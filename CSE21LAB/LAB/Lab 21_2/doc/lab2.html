CSE21 : Lab#2 - Introduction to Methods

Overview

This is the first time we will use methods to solve the problems. Methods are convenient way to encapsulate behavior that we want to use multiple times. So they can be called (invoked) anytime that is needed which is usually by another method. In fact, you have been using methods since CSE 20 because main is the starting point of our programs and a method itself. It is main's responsibility to call the correct method in the right sequence to get things started. It is important for you to do the reading first and understand how methods work before you attempt to do this lab.

Getting Started

Instructions are repeated here from your assignment page in case you unzipped the lab_21_2.zip file first and opened up lab2.html first. You can skip this section if you have properly followed the import instructions already.

We'll do things a bit differently starting this week by importing an existing eclipse project rather than creating one from scratch. By importing an already created project, all of the files will already be present including the directions in .html files.

  • Download the project archive file. Remember the location of this directory to use in step 4 to import the archive for Lab 21_2 project.
  • Start Eclipse, and select the File | Import... menu item. A dialog titled "Import" should open. (You can open this Eclipse dialog in many different ways).
  • Select General | Existing Projects into Workspace. (Don't select Archive File!). Click Next >. A dialog titled "Import Projects" should open.
  • Click the radio button for Select archive file: at the uppser left. Now, click Browse... to the right of the radio button and locate the lab_21_2.zip file that you saved earlier.
  • Having selected the archive file, you should be back in the "Import Projects" dialog. Check that you have the project "Lab 21_2" listed in the center box, and that it is selected via it's checkbox. Click Finish.

You should have a Java project in Eclipse titled Lab 21_2. This html page (that you are reading) is included in the project in the doc directory, in the file lab2.html. The Java files you will use in this lab are in the src directory, as usual.
