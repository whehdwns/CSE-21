Getting Started

You should have a Java project in Eclipse titled Lab 21_4. This html page (that you are reading) is included in the project in the doc directory, in the file lab4.html. The Java files you will use in this lab are in the src directory, as usual. If you need to add Junits into your project, then right click on it and open properties. Go to "Java Build Path", then Libraries tab. Click on "Add Library" and select JUnit then press Next>. Select Junit 4 and click Finish.
